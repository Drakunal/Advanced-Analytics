## Run Locally

Clone the project

```bash
  git clone https://github.com/Drakunal/Advanced-Analytics.git
```

Go to the project directory

```bash
  cd Advanced-Analytics
```

Opening in VS Code

```bash
  code .
```

Opening in Jupyter notebook

```bash
  jupyter notebook
```
